Python Version: Python 3


Imports: Pandas, Spotipy, Matplotlib, Seaborn, Sklearn

Installation instructions for Spotipy

First open up cmd and install Spotipy with :

pip install spotipy --upgrade

Next, Create a Spotify Developer Account
Go to "Dashboard"
Go to "Create New App"
Click on "Show Client Secret"

Copy down both "Client ID" token and "Client Secret" token

With these, past these down into the "sp" object in the top of the program code.

Now you can use Spotipy.
